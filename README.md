# 🏏 ICC Cricket World Cup 2023 – Power BI Dashboard

## 📌 Project Overview
This project is a Power BI dashboard based on the ICC Cricket World Cup 2023. It provides insights into team performance, player statistics, and match results using interactive visuals.

## 🎯 Objectives
- Analyze team and player performance  
- Track match results and statistics  
- Compare teams using key metrics  
- Build an interactive dashboard  

## 📊 Features
- Team performance (wins, losses, points table)  
- Player stats (runs, wickets, averages)  
- Match insights (results, venues)  
- Interactive filters and slicers  
- Dynamic charts and KPIs  

## 🛠️ Tools & Technologies
- Power BI  
- Excel / CSV  
- DAX  
- Power Query  

## 📂 Dataset
Includes:
- Match details  
- Player statistics  
- Team standings  

## 🚀 How to Use
1. Download the `.pbix` file  
2. Open in Power BI Desktop  
3. Use filters to explore data
